This angry birds project was made by Abhinav Mangla (2023023) and Decee Rao (2023186).

To run the project, you simply need to go to the main folder and run I using IntelliJ, it will handle all the compiling and running stuff. 

This game has 3 types of birds, 3 types of blocks and 3 types of pigs and various levels to play. It has a main menu and also a screen to change the volume from. Moreover, serialization has been done so the player's progress is saved. 
All requirements have been implemented.
